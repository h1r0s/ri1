問４　半減期と放射能から原子数を求める

炭素120gは10mol。 \(1mol=6.0\times10^{23}\)および\(^{12}C\)の同位体存在度が99%の前提より \[ N_{^{12}C} = 10 \times 6.0 \times 10^23 \times 0.99 = 6.0 \times 10^{24} \times 0.99 \] \(^{14}C\)が3.9Bqという前提より \[ \lambda N_{^{14}C} = 3.9 \] また\(^{14}C\)の半減期が\(1.8\times 10^{11}s\)という前提より \[ \lambda = \frac{0.693}{1.8\times 10^{11}} \] つまり \[ N_{^{14}C} = \frac{3.9}{\lambda} = \frac{3.9\times 1.8\times 10^{11}}{0.693} \] 上の式により \[ N_{^{14}C}/N_{^{12}C} = \frac {3.9\times 1.8\times 10^{11}} {0.693 \times 6.0 \times 10^{24} \times 0.99} \simeq 1.7\times 10^{-13} \]

問５　半減期と放射能から原子数を求める

無担体の\(^{32}P\)の放射能が\(1GBq\)ということは \[ \lambda N_{^{32}P} = 1\times 10^{9} \] また、\(T_{1/2} = 0.693/\lambda\)および \(^{32}P\)の半減期が\(1.2\times 10^6s\)の関係により \[ N_{^{32}P} = \frac{1\times 10^{9}}{\lambda} = \frac{1\times 10^{9}}{0.693/(1.2\times 10^6)} = \frac{1\times 10^{9} \times 1.2\times 10^6}{0.693} \] 上の式とアボガドロ定数\( = 6.0\times 10^{23}[mol^{-1}]\)より \[ M_{^{32}P} = \frac{32}{6.0\times 10^{23}}\times N_{^{32}P} = \frac {1\times 10^{9} \times 1.2\times 10^6 \times 32} {0.693\times 6.0\times 10^{23}} \simeq 9.2\times 10^{-8}[g] \]

問８　放射平衡について

親核種１\(\xrightarrow{\lambda_{1}}\) 娘核種２\(\xrightarrow{\lambda_{2}}\)孫娘核種３
上記のような壊変系列があるとする。 \(t = 0\)における核種１，核種２の 原子数を\(N_{1}^{0}\)、\(N_{2}^{0}\)、 \(t\)秒経過後の残存原子数を\(N_{1}\)、\(N_{2}\)、 崩壊定数を\(\lambda_{1}\)、\(\lambda_{2}\)とすると以下の式が成立する。 \[ \frac{dN_{1}}{dt} = -\lambda_{1}N_{1},\qquad \frac{dN_{2}}{dt} = \lambda_{1}N_{1} - \lambda_{2}N_{2} \tag{2.1} \] 式(2.1)をそれぞれ積分すると \[ N_{1} = N_{1}^{0}e^{-\lambda_{1}t} \tag{2.2} \] \[ N_{2} = \frac{\lambda_{1}}{\lambda_{2} - \lambda_{1}} N_{1}^{0}(e^{-\lambda_{1}t} - e^{-\lambda_{2}t}) + N_{2}^{0}e^{-\lambda_{2}t} \tag{2.3} \] 式(2.3)の最後の項\(N_{2}^{0}e^{-\lambda_{2}t}\)は 最初から存在した核種２の減衰を表すもので、 最初に娘核種が存在しない場合にはこの項は不要とる。
\(\)と\(\)の大小関係から３つの場合が考えられる。

過渡平衡\((\lambda_{1}<\lambda_{2})\)

親核種１の半減期が娘核種２の半減期に比べて長い場合、 十分な時間が経過すると式(2.3)のカッコ内の \(e^{-\lambda_{2}t}\)は\(e^{-\lambda_{1}t}\)に比べると 小さいので無視できる。
また娘核種２は最初存在しないので、 最後の項\(N_{2}^{0}e^{-\lambda_{2}t}\)は無視できる。 \[ N_{2} = \frac {\lambda_{1}} {\lambda_{2} - \lambda_{1}} N_{1}^{0}e^{-\lambda_{1}t} = \frac {\lambda_{1}} {\lambda_{2} - \lambda_{1}} N_{1} \tag{2.4} \] \[ ∴ \frac{N_{2}}{N_{1}} = \frac {\lambda_{1}} {\lambda_{2} - \lambda_{1}} \tag{2.5} \] 放射能を\(A_{1}\)、\(A_{2}\)とすると \[ \frac{A_{2}}{A_{1}} = \frac{N_{2}\lambda_{2}}{N_{1}\lambda_{1}} = 1 + \frac{N_{2}\lambda_{1}}{N_{1}\lambda_{1}} = 1 + \frac{N_{2}}{N_{1}} > 1 ,\qquad A_{2} > A_{1} \tag{2.6} \] 式(2.5)から、十分な時間が経過すると親核種と娘核種の原子数の比は一定になる。 親核種の原子数は式(2.2)により、その半減期にしたがい減少する。 一方、核種１と核種２の関係は式(2.5)より一定なので、 娘核種の原子数も親核種の半減期にしたがい減少する。 ここで十分な時間とは娘核種２の半減期の７〜１０倍を考えればよい。

永続平衡(\(\lambda_{1}\ll\lambda_{2}\))

親核種１の半減期が、娘核種２の半減期に比べて非常に長い場合、 式(2.5)の分母の\(\lambda_{1}\)は\(\lambda_{2}\)にたいし 無視できる。したがって次の式が得られる。 \[ \frac{N_{1}}{N_{2}} = \frac{\lambda_{2}}{\lambda_{1}} \quad または \quad N_{1}\lambda_{1} = N_{2}\lambda_{2} \tag{2.7} \] 十分時間が経過し永続平衡が成立すると式(2.7)左式より 核種１と核種２の比は、 \(\lambda_{2}\)と\(\lambda_{1}\)の比となり一定になる。 また式(2.7)右式より核種１の壊変率と核種２の壊変率は等しくなる。 親核種１の半減期が非常に長いので、短時間の観測では 核種１と核種２の原子数(=放射能)はほとんど変化しない。

\(\lambda_{1}>\lambda_{2}\)の場合

放射平衡は成立しない。放射平衡は成立しなくても 娘核種の原子数が極大になる時間\(t_{max}\)は次式で求められる。 \[ t_{max} = \frac{2.303}{\lambda_{2} - \lambda_{1}} \log \frac{\lambda_{2}}{\lambda_{1}} \]

問１１　化学反応

弱塩基性遊離反応

アンモニアソーダ法の工程の一部。 \(NH_{4}Cl\)から\(NH_{3}\)を取り出すのが目的。 ここでは弱塩基の塩から弱塩基の分子を取り出すために、 強塩基の\(Ca(OH)_{2}\)を加えている。 これは弱塩基性遊離反応と呼ばれる反応。
弱塩基性遊離反応とは以下のような反応。 \[ (弱塩基の塩) + (強塩基) \rightarrow (強塩基の塩) + (弱塩基) \] \(NH_{4}Cl＋Ca(OH)_{2}\)反応では、 弱塩基\(NH_{4}^{+}\)の塩\(NH_{4}Cl\)に 強塩基\(Ca^{2+}\)を加えることにより、 強塩基\(Ca^{2+}\)の塩\(CaCl_{2}\)を作り 弱塩基\(NH_{3}\)を遊離させる。 \[ 2NH_{4}Cl + Ca(OH)_{2} \rightarrow CaCl_{2} + 2NH_{3}\uparrow + 2H_{2}O \] アンモニア\(NH_{3}\)は常温常圧では無色の気体。

塩とは

酸性塩
化学式中に\(H^{+}\)をもつ塩。 硫酸水素ナトリウム\(NaHSO_{4}\)や炭酸水素ナトリウム\(NaHCO_{3}\)などが該当する。
塩基性塩
化学式中に\(OH^{-}\)をもつ塩。 塩化水酸化カルシウム\(CaCl(OH)\)や塩化水酸化マグネシウム\(MgCl(OH)\)などが該当する。
正塩
酸性塩と塩基性塩以外のすべての塩を指す。

塩基

1価の強塩基はアルカリ金属(第１族)、 2価の強塩基はアルカリ土類金属(第２族)の水酸化物のみ
弱塩基は、アンモニアか アルカリ金属(第１族)＆アルカリ土類金属(第２族)以外の金属の 水酸化物のみ！

炭酸カルシウムと塩酸

炭酸カルシウムは無色結晶または白色粉末であり、 中性の水にほとんど溶けないが、塩酸などの強酸と反応して、 二酸化炭素を放出する。 \[ CaCO_{3} + 2HCl \rightarrow CaCl_{2}+CO_{2}\uparrow + H_{2}O \]

弱酸性遊離反応

リン鉱石\(Ca_{3}(PO_{4})_{2}\)からリン肥料として使われる 過リン酸石灰\(Ca(H_{2}PO_{4})_{2}\)を生成する反応。
まず「弱酸の遊離」により、 リン鉱石のもつ\(PO_{4}^{3-}\)を 希硫酸\(H_{2}SO_{4}\)の\(H^{+}\)で \(H_{2}PO_{4}^{-}\)にする。 \[ PO_{4}^{3-} + H_{2}SO_{4} \rightarrow H_{2}PO_{4}^{-} + SO_{4}^{2-} \] 式を２倍し、 \[ 2PO_{4}^{3-} + 2H_{2}SO_{4} \rightarrow 2H_{2}PO_{4}^{-} + 2SO_{4}^{2-} \] 両辺に\(Ca^{2+}\)を３個加える。 \[ Ca_{3}(PO_{4})_{2} + 2H_{2}SO_{4} \rightarrow Ca(H_{2}PO_{4})_{2} + 2CaSO_{4} \]

\[ FeS + H_{2}SO_{4} \rightarrow FeSO_{4} + H_{2}S\uparrow \]

酸・塩基の強弱

強酸・強塩基：電離度がほぼ１の酸・塩基(水溶液中ではほぼ100%が電離)
弱酸・弱塩基：電離度が小さい酸・塩基　(水溶液中では一部のみが電離)

代表的な酸・塩基

強酸 \(HCl\)(塩酸)、\(H_{2}SO_{4}\)(硫酸)、\(HNO_{3}\)(硝酸)
弱酸 \(CH_{3}COOH\)(酢酸)、\(H_{2}S\)(硫化水素)、 \(H_{2}CO_{3}\)(炭酸)、\(H_{2}SO_{3}\)(亜硫酸)
強塩基 \(NaOH\)、\(KOH\)、\(Ca(OH)_{2}\)、\(Ba(OH)_{2}\)
(アルカリ金属の水酸化物と、アルカリ土類金属の水酸化物)
弱塩基 \(NH_{3}\)(アンモニア)
中和反応・酸化還元反応と並ぶ最重要反応原理。
三大強酸(塩酸、硫酸、硝酸)だけ覚えておけば、 あとは基本的に弱酸 (ただし\(HF\)以外のハロゲン化水素\(HBr\)、\(HI\)は強酸)。
酸・塩基の強弱で陽イオンと陰イオンの相性が異なる ことが反応原理の根幹。
強酸の\(HCl\)分子が１００あれば水溶液中では ほぼすべてが\(H^{+}\)と\(Cl^{-}\)に電離する。
一方弱酸の\(H_{2}S\)分子が１００あっても水溶液中では 数個だけが\(H^{+}\)と\(S^{2-}\)に電離する。
つまり強酸・強塩基では陽イオンと陰イオンの相性が極めて悪く、 弱酸・弱塩基では比較的相性が良い

以上の知識を元に塩\(FeS\)と強酸\(H_{2}SO_{4}\)の反応を考える。
両者を混ぜると、水溶液中には \(Fe^{2+}\)、\(S^{2-}\)、\(H^{+}\)、\(SO_{4}^{2-}\) が混在することになる。
すると４種の化合物 \(FeS\)、\(H_{2}S\)、\(FeSO_{4}\)、\(H_{2}SO_{4}\)、 が考えられる。
ここで強酸\(H_{2}SO_{4}\)の陽イオン\(H^{+}\)と 陰イオン\(SO_{4}^{2-}\)の相性は極めて悪い。
一方弱酸\(H_{2}S\)の陽イオン\(H^{+}\)と 陰イオン\(S^{2-}\)の相性は比較的良い。
すると必然的に\(H_{2}SO_{4}\)が分解して \(H_{2}S\)を生じる方向に反応が進行することになる。
つまり\(H_{2}S + H_{2}SO_{4} \rightarrow FeSO_{4} + H_{2}S\) のような反応が起こることになる。
当然ながら逆方向の反応は自然には起こらない

以上がおおまかなイメージだが厳密には以下のようになる。
まず塩\(FeS\)がどのような酸と塩基から生じるか考える
\(Fe^{2+}\)と\(S^{2-}\)に分離し、 それぞれに\(OH^{-}\)と\(H^{+}\)を結合させる。
すると\(H_{2}S + Fe(OH)_{2} \rightarrow FeS + 2H_{2}O\) のような中和反応で生じた塩であることがわかる。
\(FeS\)は弱酸\(H_{2}S\)から生じた塩であるから、 \(FeS\)は「弱酸由来の塩」といえる。
弱酸由来の塩に強酸を加えると、必然的に強酸が分解し、 弱酸が生成する。
また残りのイオンをまとめると「強酸由来の塩」が生成される。
結局、以下のような反応原理が導かれる(塩基に関しても同様)。

強酸による弱酸の遊離反応

\(弱酸由来の塩 + 強酸 \rightarrow 強酸由来の塩 + 弱酸\)
例：\(ZnS + 2HCl \rightarrow ZnCl_{2} + H_{2}S\)
例：\(CaCO_{3} + 2HCl \rightarrow CaCl_{2} + H_{2}O + CO_{2}\)
例：\(Na_{2}SO_{3} + H_{2}SO_{4} \rightarrow Na_{2}SO_{4} + H_{2}O + SO_{2}\)

強塩基による弱塩基の遊離反応

\(弱塩基由来の塩 + 強塩基 \rightarrow 強塩基由来の塩 + 弱塩基\)
例：\(NH_{4}Cl + NaOH \rightarrow NaCl + NH_{3} + H_{2}O\)

問２０　溶媒抽出の分配比

問２１　ＲＩのトレーサ利用

同位体効果

\(^{3}H\)を水素のトレーサに用いると同位体効果はない←んあこたあ、ない。 \(^{3}H\)は\(^{1}H\)の質量の３倍なので、むしろ同位体効果は大きい。真逆
同位体効果は大きくわけて二種類あり、 質量に起因するものと核の形状や電荷分布が 異なることに起因するものがある。
前者を質量効果といい、物理的・化学的性質が異なり、 原子番号が小さい同位体ほど顕著になり、 水素の同位体である重水素やトリチウムで最大となる。
後者を体積効果といい、核の体積・形・電荷分布 の違いにより外部の電子状態に影響を与え、 こちらは逆にウランのような原子番号が大きな 同位体ほど顕著である。
同位体効果は物理的性質はもとより 化学反応の速度などにも影響を与える。 分子分光学における同位体効果は 同位体存在比の測定に用いられる。
このような同位体効果を用いることによって 重水素やウランを分離するなどといった 同位体の分離が可能となる。

同位体交換反応

用いるRIの同位体交換反応速度が大きいことが必要である ←んなこたあ、ない
同位体交換反応とは、 異種同位体（XとX*）からなる2種類の化合物（AXとBX*）を 接触させた時に化合物間で同位体の交換が起こる反応 （AX＋BX*→AX*＋BX）をいう。 この交換反応は、化学的に同一挙動である （同程度の逆反応が起こる）と考えられるが、 実際には同位体の質量差に起因した結合エネルギーの 相違により反応速度や平衡定数に僅かな差ができる。 この僅かな差を多段に組み合わせる分離法を同位体交換反応法という。 また、化合物の存在量の分析に同位体交換反応を利用する方法も 同位体交換反応法（同位体希釈分析）という。　
トレーサとして用いるRIは、同位体交換速度が、 実験期間中、無視できるほど小さくなければならない。

放射線効果

比放射能の高い標識化合物を用いるときは、自己放射線分解に注意する。←そのとおり
放射線効果によってひき起される物理的、化学的、生物学的変化を放射線効果という。 一般に、無機化学反応では放射線による影響は少なく、10Gy程度から 水の放射分解に伴う酸化、還元、分解が起こりはじめる。 したがって線量の低い通常のトレーサ実験では、放射線効果は考慮しなくてよい。 有機反応でも、同様に通常のトレーサ実験では考慮しなくてもよい。 ただ比放射能の高い標識化合物は、それ自身が放射線効果により 分解している恐れがあるので、その放射化学純度を調べる必要がある。

トレーサの原子価

用いるRIの原子価を、対象とするイオンの原子価に合わせる←そのとおり
原子価（げんしか）とはある原子が何個の他の原子と結合するかを表す数である。 元素によっては複数の原子価を持つものもあり、 特に遷移金属は多くの原子価を取ることができるため、 多様な酸化状態や反応性を示す。
酸化状態が１つしかない\(^{24}Na\)や\(^{42}K\)のような場合は問題ないが、 たとえば\(I\)とか\(S\)の化合物などのように、様々な酸化状態をもつ元素は、 トレーサの酸化状態と、トレースされる非RIの酸化状態に留意する必要がある。 ヨウ素酸ナトリウム(\(NaIO_{3}\))の追跡において ヨウ化ナトリウムをトレーサに用いる場合や、 硫酸(\(H_{2}SO_{4}\))の追跡において硫化水素や亜硫酸を トレーサに用いる場合は、トレーサとトレーシーが異なる酸化状態にあるので 両者の酸化状態を揃えるために、適切な酸化剤や還元剤を加える必要がある。

問２２　同位体希釈法

  1. 特徴
    1. 混合物中のある成分を定量するとき、 その成分だけを完全に分離して定量しなければ ならないが、化学的性質が似ていて完全に分離できないもの (希土類元素、アミノ酸、抗生物質、ステロイドなど) が定量できる
    2. 目的成分を完全に分離することなく、どの一部を純粋に 取り出すことができれば定量できる。
  2. 欠点
    1. ラベル付き化合物をつくる必要がある。
    2. ラベル付き化合物のRIが同位体交換反応により、 定量すべき化合物以外に移るときは使用できない。

直接期釈放

混合物中の定量すべき試料の重量\(X\)を定量するため、 ラベル付き化合物の一定量 (重量\(a\)、放射能\(A_{S}^{*}\)、比放射能\(S_{0}=A_{S}^{*}/a\)) を加えて十分に混合し、その中から一定量を取り出す。 取り出された化合物の重量\(W\)と放射能\(A\)を測定して 比放射能\(S=A/W\)を求める。
混合前のラベル付き化合物の全放射能は\(S_{0}a\)、 混合物の全放射能は\(S(a+X)\)で、全放射能は混合の前後で等しいので、 \[ S(a + X) = S_{0}a \] これを変形すると、試料の重量は \[ X = a \left( \frac{S_{0}}{S} - 1 \right) \] この式は\(X \gg a\)、つまりラベル付き化合物の重量が十分小さいときは 以下のように変形できる。 \[ X = \frac{S_{0}}{S}a \]

逆希釈法

定量すべき化合物が放射性で、その比放射能が既知の場合、 逆希釈でその化合物の重量を定量できる。 直接希釈法とは逆に非RIの化合物を添加して、 ラベル付き化合物を定量する。 定量すべき試料中の化合物の比放射能を\(S_{0}\)、 重量を\(X\)とし、これに非RIである同じ化学形の一定量 (重量\(a\))を加えて十分に混和する。 この化合物から定量しようとする目的物を取り出し、 その比放射能を\(S\)とすれば、 混合前の全放射能は\(S_{0}X\)、 混合後の全放射能は\(S(X + a)\)となる。
全放射能は混合の前後で等しいので \[ S_{0}X = S(X + a) \] これを変形して \[ X = a \left( \frac{S}{S_{0} - S} \right) \] この方法の難点は、試料中に最初に存在するラベル付き化合物の 比放射能があらかじめ判っていなければならないことである。

二重希釈法

逆希釈法は比放射能\(S_{0}\)が既知の試料に限られるが、 二重希釈法はそれが未知でも使用できる。 原理は、試料から等しい量(または試料を二等分)をとり、 それぞれにたいして異なる重量の非RI化合物\(a_{1}\)、\(a_{2}\) を加えて混和し、それぞれの化合物から一部を分離し、 その重量と放射能を測定することで、比放射能\(S_{1}\)、\(S_{2}\) を求める。
等量な１試料中に含まれている、当量すべき化合物の重量を\(X\)、 比放射能を\(S_{0}\)とすると、以下の連立方程式が成り立つ。 \[ S_{0}X = S_{1}(X + a_{1}) \tag{9.2} \] \[ S_{0}X = S_{2}(X + a_{2}) \tag{9.3} \] 式(9.2)と式(9.3)は等しいことから \[ X = \frac{S_{2}a_{2} - S_{1}a_{1}}{S_{1} - S_{2}} \tag{9.4} \]

アイソトープ誘導体法(isotope derivative method)

標識化合物が得られない場合は、直接希釈法で非放射性物質は 定量できないが、ある種の化合物についてはアイソトープ誘導体法で定量できる。 直接希釈法が適用できる場合でも、アイソトープ誘導体法を用いれば 複雑な標識化合物でなく簡単な標識試薬により注目化合物が定量できる。
化学的性質がよく似た物質A、B、C、…の混合物について、 Aを定量したい場合、これらの物質と結合して誘導体を与えるような 放射性試薬（R*）を試料に加えて反応させ、 AR*、BR*、CR*…のような放射性誘導体を作りる。 逆希釈法に基づいて非放射性のARの一定量を加え、 AR* + ARの一部を純粋に分離して比放射能を測定すれば、 \(W_{x}\)からR*の量、したがってAの量を求めることができまる。
［例］アミノ酸は塩化p-ヨードフェニルスルホニルとピプシル誘導体を作りやすいので、 タンパク質の加水分解生成物中のアミノ酸の組成を調べる
